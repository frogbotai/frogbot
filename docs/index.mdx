Write a `frogbot.config.ts` and instantly get a hosted UI, agent runtime, chat, skills, flows, integrations, model routing, file storage, authentication, REST and GraphQL APIs, and a job system — all in one MIT open-source TypeScript codebase you own and deploy anywhere.

## What it gives you

- An AI workspace UI for chats, agents, skills, flows, and connections
- A file-based defaults system for skills, agents, MCPs, and flows
- A pieces system for integrations (Slack, Gmail, Notion, GitHub, and more)
- A piece connector layer that handles managed OAuth without per-piece OAuth apps
- A job system for triggers, polling, scheduled flows, and waitpoints
- Web, desktop, and browser extension clients sharing the same React app shell
  - A plugin system that extends the platform through config transforms
