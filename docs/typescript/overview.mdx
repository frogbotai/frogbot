FrogBot supports TypeScript natively, and not only that, the entirety of the CMS is built with TypeScript. To get started developing with FrogBot and TypeScript, you can use one of FrogBot's built-in boilerplates in one line via `create-frogbot-app`:

```bash
npx create-frogbot-app@latest
```

Pick a TypeScript project type to get started easily.

## Setting up from Scratch

It's also possible to set up a TypeScript project from scratch. We plan to write up a guide for exactly how—so keep an eye out for that, too.
